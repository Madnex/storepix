# Default Template

Gradient background with decorative blur effects and a centered device mockup.

## Layout

```
┌─────────────────────────┐
│    ○ blur   ○ blur      │
│                         │
│       Headline          │
│       Subheadline       │
│                         │
│      ┌─────────┐        │
│      │  ┌───┐  │        │
│      │  │   │  │        │
│      │  │ S │  │        │
│      │  │ C │  │        │
│      │  │ R │  │        │
│      │  │ E │  │        │
│      │  │ E │  │        │
│      │  │ N │  │        │
│      │  │   │  │        │
│      │  └───┘  │        │
│      └─────────┘        │
│           ○ blur        │
└─────────────────────────┘
```

## Features

- Gradient background with animated-style blur circles
- Headline + subheadline text above device
- Device frame with notch/Dynamic Island/home button
- Optional status bar overlay
- Light/dark theme support
- `layout: 'bottom'` moves text below device

## Best For

- App Store marketing screenshots
- Eye-catching promotional graphics
- Apps that benefit from colorful, dynamic backgrounds
