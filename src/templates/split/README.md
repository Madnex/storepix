# Split Template

Side-by-side layout with text on one half, device on the other.

## Layout

```
┌─────────────────────────┐
│            │            │
│            │ ┌────────┐ │
│            │ │ ┌────┐ │ │
│  Headline  │ │ │    │ │ │
│            │ │ │ SC │ │ │
│ Subheadline│ │ │ RE │ │ │
│            │ │ │ EN │ │ │
│            │ │ │    │ │ │
│            │ │ └────┘ │ │
│            │ └────────┘ │
│            │            │
└─────────────────────────┘
```

## Features

- 50/50 split between text and device
- Large text area for detailed messaging
- Device frame with notch/Dynamic Island/home button
- Optional status bar overlay
- Light/dark theme support
- `layout: 'right'` moves text to the right side

## Best For

- iPad screenshots (landscape-friendly)
- When you need more room for marketing copy
- Feature explanations with detailed text
- B2B or productivity apps
