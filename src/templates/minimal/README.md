# Minimal Template

Clean solid-color background with a centered device mockup.

## Layout

```
┌─────────────────────────┐
│                         │
│       Headline          │
│       Subheadline       │
│                         │
│      ┌─────────┐        │
│      │  ┌───┐  │        │
│      │  │   │  │        │
│      │  │ S │  │        │
│      │  │ C │  │        │
│      │  │ R │  │        │
│      │  │ E │  │        │
│      │  │ E │  │        │
│      │  │ N │  │        │
│      │  │   │  │        │
│      │  └───┘  │        │
│      └─────────┘        │
│                         │
└─────────────────────────┘
```

## Features

- Solid color background (customizable via theme)
- Headline + subheadline text above device
- Device frame with notch/Dynamic Island/home button
- Optional status bar overlay
- Light/dark theme support
- `layout: 'bottom'` moves text below device

## Best For

- Clean, professional look
- Apps with strong brand colors
- When you want the screenshot to be the focus
- Minimalist design aesthetic
